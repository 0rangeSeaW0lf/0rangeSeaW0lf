# José Moreno

Investment partner by day, systems-builder on the edges. Based internationally, operating across Latin America, Europe, and the Gulf.

I've been on GitHub since 2011. Between 2014 and 2017 I built full-stack web applications in Ruby on Rails and JavaScript / AngularJS across several threads: day roles at British American Tobacco (SAP-side) and wetransform (GIS front-end), mentoring web-development students at Udacity, and technical-lead work on my own early-stage ventures, notably TRIBÙ (a hospitality-deals platform built during the IESE Summer Entrepreneurship Experience) and Mandadito (a cross-border commerce concept between Latin America and Iberia). In 2017 I moved to investing full-time as a founding partner at a search-fund aggregator; the next nine years were cap tables and compounders rather than commits. I've been building again for about a year now, at a different layer: agentic LLM workflows and the systems design required to run an analytical pipeline across 60+ skills, hooks, and evaluation gates as part of my investment work.
